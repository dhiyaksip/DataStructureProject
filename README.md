# Data Structure Project - Group 2

## Deskripsi
Project ini mengimplementasikan:
- Queue (Sistem Antrean)
- Stack (Undo/Redo)
- Linked List (Data Mahasiswa)

## Cara Menjalankan
1. Compile semua file:
   javac *.java

2. Jalankan:
   java MainApp

## Struktur Data
- Queue → FIFO
- Stack → LIFO
- Linked List → Dynamic Data

## Analisis
Queue cepat untuk antrean,
Stack cocok untuk undo/redo,
Linked List fleksibel dalam manipulasi data.
